## Scope
Refactoring `src/` modules while maintaining behavior.

## Process
1.  **Establish Baseline**: Ensure existing tests pass.
2.  **Small Steps**: Make incremental changes.
3.  **Test-Driven**: Update tests if needed, but ensure core logic remains same.
4.  **Verify**: Run all tests before finalizing.

## Output Format
- Description of refactor goal.
- Summary of changes.
- Proof of verification.
